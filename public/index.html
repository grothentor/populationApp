Вычисление популяции, P = P0ekt

Начальная популяция и прирощение популяции должны быть числами.
Для решения задачи необходимо чтобы целевое значение или время было задано числами.
Начало временного промежутка должно быть больше его конца.
Для получения решения необходимо задать корректные значения для начальной популяции, прироста популяции. За тем необходимо указать необходимое целевое значение популяции или(и) промежуток времени, на котором будет отслеживаться популяций. При введении корректнного целевого значения популяции будет отображено через какое время будет достигнута такая популяция. При указании промежутка времени будет отображен график с изминением числа популяции на нем.
